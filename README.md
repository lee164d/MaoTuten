## 工作流程

本项目采用 GitHub Issues + PR 工作流管理内容生产决策：

- **Issue** = 认知修正 / 流程问题 / 待评估事项
- **PR** = 经过讨论确认后的文档修改
- **Merge** = 人类决策者确认，变更生效
