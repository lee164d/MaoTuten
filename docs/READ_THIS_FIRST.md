# READ_THIS_FIRST

本文件用于帮助新人 / 新AI 在最短时间内理解猫图腾项目。

## 你需要先知道什么
猫图腾不是普通宠物号，也不是单纯 AI 出图项目。
它是一个以“情绪方向”为驱动的猫 IP 短视频生产系统。

## 最短阅读顺序
1. 《猫图腾_生产线手册_V1.0》
2. 《猫图腾_视觉叙事规范_V1.0》
3. 《AI创意生产协作系统手册_V1.0》
4. 按你的角色进入对应执行框架
5. 扫描 `PENDING_EVALUATION.md`
